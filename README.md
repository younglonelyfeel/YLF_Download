# YLF Downloader

Tool tải video đa nền tảng (YouTube, TikTok, Facebook...) hỗ trợ tự động copy caption.

## Tính năng
- Tải video chất lượng cao (MP4).
- Tự động copy tiêu đề/caption.
- Chế độ Mini Mode gim trên màn hình.
- Tự động cập nhật (đối với bản cũ).

## Cách sử dụng
1. Tải file `YLF Downloader.zip` từ mục Releases.
2. Giải nén ra thư mục.
3. Chép file `ffmpeg.exe` vào cùng thư mục.
4. Chạy file `YLF Downloader.exe`.
